# XFly - 消耗货币飞行系统

一个基于 Paper 1.21 的飞行插件，通过 Vault 经济系统按分钟扣费。

## 前置依赖

- **Vault**（必需）
- **PlaceholderAPI**（可选）

## 命令

| 命令 | 描述 | 权限 |
|------|------|------|
| `/xfly` | 切换飞行模式 | xfly.use |
| `/f` | 同上（别名） | xfly.use |
| `/xfly gui` | 打开飞行控制 GUI | xfly.use |
| `/xfly reload` | 重载配置文件 | xfly.reload |

## 权限

| 权限节点 | 描述 | 默认 |
|----------|------|------|
| `xfly.use` | 允许使用飞行功能 | 所有人 |
| `xfly.bypass` | 绕过消耗，无限飞行 | OP |
| `xfly.reload` | 重载配置文件 | OP |

## 功能特性

- **预付制计费**：开启飞行时预付1分钟费用，之后每满1分钟扣费一次
- **余额不足警告**：余额不足时给予10秒缓冲期，动作栏+消息提示，超时自动关飞
- **悬停暂停计费**：玩家在空中不动超过5分钟暂停计费，移动后恢复
- **消耗阈值提醒**：GUI 中可设置单次飞行消耗阈值（50/100/150/200），达到后提醒并自动关飞
- **GUI 控制界面**：可视化操作，显示余额、费用说明、飞行状态
- **可配置消息**：所有提示文本均可在 config.yml 中自定义

## 配置

详见 `config.yml`，支持颜色代码 `&`。
